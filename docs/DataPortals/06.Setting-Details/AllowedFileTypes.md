# AllowedFileTypes

The **AllowedFileTypes** column on a container restricts the acceptable extensions of uploaded files. Supports comma-separated lists.

### Example Usage

| AllowedFileTypes                | Description                                                  |
| ------------------------------- | ------------------------------------------------------------ |
| pdf,docx,txt,xlsx               | Only PDFs, Word Documents, Text files, and Excel files can be uploaded |

## Controls

The **AllowedFileTypes** column applies to the following controls:

- [File](../05.Control-Details/File.md)
